# Conclusão sobre Tipos de Dados

Conforme o _ECMAScript_ (Orgão regulamentador do JS) standard temos 9 tipos de dados

* Data Types
    * Primitive / Primitive value
    * Structural
    * Structural Primitive

## Primitivos

<br>

* *IMPORTANTE*: São valores imutáveis

<br>

* String -> "AB" != "ab" != ...
* Number -> 10 != 21 != ...
* Boolean -> true != false
* undefined
* Symbol
* BigInt

## Estruturais

* Object
    * Array
    * Map
    * Set
    * Date
    * ...

* Function
