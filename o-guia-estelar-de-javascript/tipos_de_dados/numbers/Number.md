# Numbers

* 33 = inteiro
* 12.5 = real (números flutuntes)
* NaN (Not a Number) = não é um número
* Infinity = infinito

<br>

* Exemplos
    * Inteiros
        ```js
        console.log(33)
        // OU
        console.log(-33)
        ```

    * Real
        ```js
        console.log(12.5 * 33)
        // OU
        console.log(12.5 / 33)
        ```

    * NaN
        ```js
        console.log(12.5 / asdwa) // Erro: NaN
        ```

    * Infinity
        ```js
        console.log(12.5 === Infinity) // Verificação se o número é infinito
        ```
