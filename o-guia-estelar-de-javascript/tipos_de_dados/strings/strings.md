# Strings

* Cadeia de caracteres
    * "" = aspas duplas
    * '' = aspas simples
    * `` = template literals ou template strings

* Exemplos
    * Aspas duplas
        ```js
        console.log("Gabriel")
        ```
    * Aspas simples
        ```js
        console.log('"Gabr"iel"')
        ```
    * Template strings
        ```js
        console.log(`
        "'

        Gabriel
        '''"
        `)

        console.log(`Gabriel ${1 + 1}`) // OBS: interpolação = colocar expressões dentro de template strings
        ```
